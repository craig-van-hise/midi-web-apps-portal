# Product Requirements Prompt (PRP) — ✅ Completed

## 1. Project Context & Objectives

**Objective:** Resolve critical audio playback regressions in the `midi-tonnetz` plugin regarding polyphonic hardware input, hanging notes, and voice-leading modes.
**Current State:** The initial implementation introduced state race-conditions causing dropped polyphonic MIDI notes, unintended octave jumps on hardware input, identical behavior for both voice-leading algorithms, and hanging notes due to timeout-based sustain.
**Outcome:** The plugin will accurately capture polyphonic hardware chords without applying algorithmic shifts. UI buttons (Nav Arrows, Play) will be refactored into momentary switches (`onPointerDown`/`onPointerUp`) to provide precise articulation cutoffs. The "Parallel" and "Parsimonious" algorithms will be mathematically decoupled to ensure distinct structural behaviors.

## 2. Technical Decisions & Dependencies

* **Polyphonic State Integrity:** Hardware MIDI events must directly mutate `activeMidiNotesRef` without passing through any "Gravity" or "Voice Leading" logic.
* **Momentary Actuation:** Remove all `setTimeout` note-off logic. Bind Note On to `onPointerDown` and Note Off to `onPointerUp`/`onPointerLeave`.
* **Algorithmic Divergence:** * **Parallel (Index-Mapped):** Sorts both arrays. Compares Target Index `[i]` *only* to Previous Index `[i]`.
* **Parsimonious (Shortest Path):** Compares Target to *all* Previous notes and picks the absolute shortest distance, regardless of structural position.

### Phase 3: Strict Algorithmic Divergence (Voice Leading) — ✅ Completed

* **Objective:** Ensure Parallel and Parsimonious modes yield musically distinct results.
* **Instructions:**
1. **Implement Parallel (Index-Mapped):**
```javascript
const calculateParallel = (targetPcs, prevNotes) => {
  const sortedPrev = [...prevNotes].sort((a, b) => a - b);
  const sortedPcs = [...targetPcs].sort((a, b) => a - b);
  return sortedPcs.map((pc, i) => {
    const anchor = sortedPrev[i % sortedPrev.length]; // Strict 1:1 mapping
    let best = pc, minDiff = Infinity;
    for (let oct = -2; oct <= 10; oct++) {
      const diff = Math.abs((pc + oct * 12) - anchor);
      if (diff < minDiff) { minDiff = diff; best = pc + oct * 12; }
    }
    return best;
  });
};

```


2. **Implement Parsimonious (Shortest Path):**
```javascript
const calculateParsimonious = (targetPcs, prevNotes) => {
  return targetPcs.map(pc => {
    let best = pc, minDiff = Infinity;
    for (let oct = -2; oct <= 10; oct++) {
      const candidate = pc + oct * 12;
      // Compares against ALL previous notes, finding the absolute closest
      for (const prev of prevNotes) {
        const diff = Math.abs(candidate - prev);
        if (diff < minDiff) { minDiff = diff; best = candidate; }
      }
    }
    return best;
  });
};

```
