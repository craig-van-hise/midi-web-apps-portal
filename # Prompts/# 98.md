## 1. Project Context & Objectives
The `transposeDiatonically` function currently relies on a "closest-MIDI" algorithm because it is agnostic to the enharmonic spelling of the note being transposed. This causes incorrect outputs when transposing non-diatonic chords (e.g., Bb and Eb shifting to G and C instead of A and D). The objective is to upgrade the algorithm to accept a `currentSpelling` string. The function will extract the absolute diatonic step based on the letter class, apply the delta, and map it securely to the target scale's pitch class.

## 2. Technical Decisions & Dependencies
* **Target Files:**
  * `src/plugins/chord-notator/utils/notationMath.ts`
  * `src/plugins/chord-notator/components/NotationCanvas.tsx`
* **Logic Specification:** 1. Extract the leading letter (A-G) from the spelling string.
  2. Map the letter to a 0-6 diatonic step (0=C, 1=D, etc.).
  3. Calculate the absolute scale step accounting for the current MIDI octave.
  4. Apply the transposition delta (+1, -1, etc.).
  5. Search the target key's `diatonicMap` to find the exact pitch class associated with the new step.

    3. Insert the following spelling-aware logic immediately below `const diatonicMap = getDiatonicMap(keySignature, lut);`:
       ```typescript
       if (currentSpelling) {
           const letterMatch = currentSpelling.match(/^[A-G]/);
           if (letterMatch) {
               const letter = letterMatch[0];
               const step = ["C", "D", "E", "F", "G", "A", "B"].indexOf(letter);

               if (step !== -1) {
                   const pitchClass = currentMidi % 12;
                   let baseOctave = Math.floor(currentMidi / 12) - 1;

                   // Boundary correction for C/B splits
                   const targetLetterPC = [0, 2, 4, 5, 7, 9, 11][step];
                   if (pitchClass === 11 && targetLetterPC === 0) baseOctave += 1;
                   if (pitchClass === 0 && targetLetterPC === 11) baseOctave -= 1;

                   const absoluteStep = (baseOctave * 7) + step;
                   const newAbsoluteStep = absoluteStep + delta;

                   const newOctave = Math.floor(newAbsoluteStep / 7);
                   const newStep = ((newAbsoluteStep % 7) + 7) % 7;

                   let targetPC = -1;
                   for (const [pc, data] of diatonicMap.entries()) {
                       if (data.step === newStep) {
                           targetPC = pc;
                           break;
                       }
                   }

                   if (targetPC !== -1) {
                       let finalOctave = newOctave;
                       const newTargetLetterPC = [0, 2, 4, 5, 7, 9, 11][newStep];
                       if (targetPC === 11 && newTargetLetterPC === 0) finalOctave -= 1;
                       if (targetPC === 0 && newTargetLetterPC === 11) finalOctave += 1;

                       const targetMidi = (finalOctave + 1) * 12 + targetPC;
                       return Math.max(0, Math.min(127, targetMidi));
                   }
               }
           }
       }
       ```
