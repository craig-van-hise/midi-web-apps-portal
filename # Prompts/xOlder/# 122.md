# Product Requirements Prompt (PRP) — Sequence Keyswitch strict Cleanup ✅

## 1. Project Context & Objectives
We need to enforce a strict state dependency rule between the `StepSequencer` timeline and the active `Keyboard` sequence keyswitches: **A keyswitch must be removed if its corresponding step on the timeline is empty.** This ensures that when the user clicks the "Clear all steps" (Trash Can) button, all sequence keyswitches are instantly cleared. It also future-proofs the app so that if an individual step is ever cleared, its specific keyswitch is immediately pruned from the keyboard.

## 2. Technical Decisions & Dependencies
* **Target File:** `src/plugins/chord-notator/components/StepSequencer.tsx`.
* **State Management:** We will extract `sequenceKeyswitches` and `setSequenceKeyswitches` from the `useMidi()` context.
* **Reactive Pruning:** We will implement a `useEffect` that monitors the `sequence` state. If any key in the `sequenceKeyswitches` map points to a step index where `sequence[stepIndex].notes.length === 0`, that keyswitch is deleted from the state.
* **Explicit Clear:** For instant UI response, we will also explicitly clear the `sequenceKeyswitches` state in the Trash Can's `onClick` handler.

## 3. Implementation Phases

### Phase 1: Reactive Keyswitch Pruning (`StepSequencer.tsx`) — ✅
* **Objective:** Implement a `useEffect` hook that automatically removes keyswitches mapped to empty sequencer steps.
* **Instructions:**
  1. Ensure `sequenceKeyswitches` and `setSequenceKeyswitches` are destructured from `useMidi()`.
  2. Add the following `useEffect` below the `sequence` state declaration:
     ```javascript
     useEffect(() => {
       if (!sequenceKeyswitches || Object.keys(sequenceKeyswitches).length === 0) return;

       let hasChanges = false;
       const updatedSwitches = { ...sequenceKeyswitches };

       Object.entries(updatedSwitches).forEach(([midiNote, stepIndex]) => {
         if (!sequence[stepIndex] || !sequence[stepIndex].notes || sequence[stepIndex].notes.length === 0) {
           delete updatedSwitches[midiNote];
           hasChanges = true;
         }
       });

       if (hasChanges) {
         setSequenceKeyswitches(updatedSwitches);
       }
     }, [sequence, sequenceKeyswitches, setSequenceKeyswitches]);
     ```
