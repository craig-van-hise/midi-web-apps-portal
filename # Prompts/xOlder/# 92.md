

# Product Requirements Prompt (PRP) — ✅ Completed

## 1. Project Context & Objectives

The previous attempts to illuminate active keyswitches failed because the global `__MIDI_INTERCEPTOR` in `MIDIProvider.tsx` swallows raw MIDI events for mapped keyswitches. Consequently, `Keyboard.tsx` never receives the raw `noteOn` data. The objective is to discard the broken raw MIDI tracker and instead listen to the semantic `APP_BUTTON_PRESS_ON` and `APP_BUTTON_PRESS_OFF` events broadcasted by the provider, mapping them to physical keys to trigger the 100% semantic color flare.

## 2. Technical Decisions & Dependencies

* **Event Bridging:** `Keyboard.tsx` will listen to `APP_BUTTON_PRESS_ON/OFF`, mapping the incoming `buttonId` to a `midiNote` using the `buttonConfigs` context.
* **State Protection:** A local `activeKeyswitches` Set (via `useRef`) will track held keyswitches to protect them from being overwritten when `NotationCanvas` broadcasts a global `refresh` sweep.

React.useEffect(() => {
    const handleBtnOn = (e: any) => {
        const buttonId = e.detail?.buttonId;
        const config = buttonConfigs[buttonId];
        if (config && config.midiNote !== -1) {
            activeKeyswitches.current.add(config.midiNote);
            updateKeyVisuals88(config.midiNote, 'ks_active');
        }
    };
    const handleBtnOff = (e: any) => {
        const buttonId = e.detail?.buttonId;
        const config = buttonConfigs[buttonId];
        if (config && config.midiNote !== -1) {
            activeKeyswitches.current.delete(config.midiNote);
            updateKeyVisuals88(config.midiNote, '');
        }
    };

    window.addEventListener('APP_BUTTON_PRESS_ON', handleBtnOn);
    window.addEventListener('APP_BUTTON_PRESS_OFF', handleBtnOff);
    return () => {
        window.removeEventListener('APP_BUTTON_PRESS_ON', handleBtnOn);
        window.removeEventListener('APP_BUTTON_PRESS_OFF', handleBtnOff);
    };
}, [buttonConfigs]);

```

export const updateKeyVisuals88 = (note: number, color: string) => {
    const el = document.getElementById(`pk88-${note}`);
    if (!el) return;

    const isBlack = [1, 3, 6, 8, 10].includes(note % 12);
    const ksColor = el.dataset.ksColor;

    let finalColor = color;
    if (color === 'ks_active' && ksColor) {
        finalColor = ksColor;
    } else if (ksColor && color && color !== '') {
        finalColor = ksColor; // Always override standard active colors with ksColor
    }

    if (finalColor && finalColor !== '') {
        // ACTIVE STATE
        el.style.backgroundColor = finalColor;
        el.style.boxShadow = `inset 0 -5px 10px rgba(0,0,0,0.1), 0 0 12px ${finalColor}`;
        if (isBlack) {
            el.style.zIndex = '11';
            el.style.borderBottom = '1px solid #000';
            el.style.borderLeft = '1px solid #333';
            el.style.borderRight = '1px solid #333';
            if (ksColor) el.style.borderTop = `2px solid ${ksColor}`;
        }
    } else {
        // IDLE STATE
        if (ksColor) {
            el.style.backgroundColor = isBlack 
                ? `color-mix(in srgb, ${ksColor} 60%, #444444)` 
                : `color-mix(in srgb, ${ksColor} 45%, #ffffff)`;
            el.style.borderTop = `2px solid ${ksColor}`;
        } else {
            el.style.backgroundColor = isBlack ? '#444' : '#fff';
            el.style.borderTop = 'none';
        }
        el.style.boxShadow = 'none';
        if (isBlack) {
            el.style.zIndex = '10';
            el.style.borderBottom = '6px solid #000';
            el.style.borderLeft = '1px solid #333';
            el.style.borderRight = '1px solid #333';
        }
    }
};

```
