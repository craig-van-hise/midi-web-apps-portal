
# Product Requirements Prompt (PRP) — 🔵 Active

## 1. Project Context & Objectives

The application is currently failing to produce audio due to incomplete initialization of the Tone.js/smplr audio engine and strict browser autoplay/MIDI permission policies. The primary objective is to bind `audioEngine.init()` and `navigator.requestMIDIAccess()` to an explicit user gesture within `App.jsx`, ensuring the `AudioContext` is properly resumed and MIDI permissions are requested securely.

## 2. Technical Decisions & Dependencies

* **Architecture Pattern:** Introduce a global "Boot" state (`hasStarted`) in `App.jsx`.
* **Audio Autoplay Compliance:** `audioEngine.init()` must be triggered asynchronously via a user `onClick` or `onKeyDown` event.
* **Web MIDI Security Compliance:** Defer the `navigator.requestMIDIAccess()` call until *after* the user gesture to prevent Chrome from blocking the permission request.
* **UI/UX:** Implement a transient full-screen overlay or bind the initialization to a global window listener that removes itself after the first interaction.

## 3. Implementation Phases

### Phase 1: Engine Initialization & Gesture Binding — ✅ Completed

* **Objective:** Ensure the Tone.js AudioContext and the internal `AudioEngine` graph are built upon the first user interaction.
* **Instructions:** 1. In `src/core/App.jsx`, import `audioEngine` if not already imported, and create a new boolean state: `const [hasStarted, setHasStarted] = useState(false);`.
2. Create an initialization function `handleAppStart` that marks `hasStarted` to true and calls `await audioEngine.init()`.
3. Render a full-screen, visually appealing "Click Anywhere to Start" overlay `div` (using absolute positioning, high z-index, and backdrop-blur) that displays when `!hasStarted`. Attach `handleAppStart` to its `onClick` handler.

### Phase 2: Deferred MIDI Permission Request — 🔘 Pending

* **Objective:** Move the Web MIDI permission request out of the initial mount cycle and into the user-triggered boot sequence.
* **Instructions:**
