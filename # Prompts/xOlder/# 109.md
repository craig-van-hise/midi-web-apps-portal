# Product Requirements Prompt (PRP) — 🔘 Pending

## 1. Project Context & Objectives
Upgrade the `StepSequencer` component to make the 8 mini chord cards fully interactive. Clicking a chord card must trigger four synchronized actions:
1. **Audio Playback:** Instantly play the chord as a 500ms audio preview using the global `audioEngine`.
2. **Card Selection:** Add a persistent active visual highlight to the clicked chord pill card.
3. **Notation Staggering:** Change the color of the mini-noteheads and accidentals within that specific bar to purple (`#aa3bff`) to match the app's selection aesthetic.
4. **Main Canvas Sync:** Inject the bar's note dataset directly into the main `NotationCanvas` context so it populates the main grand staff layout instantly.

## 2. Technical Decisions & Dependencies
* **Target File:** `src/plugins/chord-notator/components/StepSequencer.tsx`
* **State Hook:** Introduce a `selectedStep` state (`number | null`) to manage the actively selected measure index.
* **State Injection API:** Destructure `updateActiveNotes` and `uiVelocity` from `useMidi()` to pass selected sequence arrays smoothly back up to the global context bus.
