### Step 2: Build the Audio Consumer Loop

* Still in `src/core/App.jsx`, create a dedicated `useEffect` that establishes a high-frequency polling loop using `requestAnimationFrame`. This loop will continuously check the `MidiRingBuffer` and trigger the Master Rompler.
* Add this logic:
```javascript
useEffect(() => {
  let animationFrameId;

  const drainBuffer = () => {
    // Pull all available messages from the buffer this frame
    let data = midiRingBuffer.pull();
    while (data) {
      const [status, note, velocity] = data;
      const command = status & 0xf0;

      // Route directly to the Rompler window hooks
      if (command === 0x90 && velocity > 0) {
        if (window.playNoteOn) window.playNoteOn(note, velocity);
      } else if (command === 0x80 || (command === 0x90 && velocity === 0)) {
        if (window.playNoteOff) window.playNoteOff(note);
      }

      // Fetch next message in the queue, if any
      data = midiRingBuffer.pull(); 
    }
    animationFrameId = requestAnimationFrame(drainBuffer);
  };

  // Start the loop
  drainBuffer();

  return () => cancelAnimationFrame(animationFrameId);
}, []);

```
