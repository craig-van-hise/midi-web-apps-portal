# Product Requirements Prompt (PRP) — ✅ Completed

## 1. Project Context & Objectives
The visual bounds of the `NotationCanvas` need to be tightened to improve readability for chords spanning extreme registers. The objective is to lower the thresholds for applying octavation lines. 
* `8va` will trigger for notes above F6.
* `15ma` will trigger an octave above that (G7).
* `8vb` will trigger for notes below C2.
* `15mb` will trigger an octave below that (B0).
Additionally, the plain-text labels ("8va", "8vb") will be upgraded to utilize the official SMuFL glyphs rendered via the Bravura font.

## 2. Technical Decisions & Dependencies
* **Target Files:**
  * `src/plugins/chord-notator/utils/notationMath.ts` (SMuFL Dictionary)
  * `src/plugins/chord-notator/components/NotationCanvas.tsx` (Layout Engine)
* **SMuFL Glyphs:**
  * `\uE510` (ottavaAlta) for 8va
  * `\uE514` (quindicesimaAlta) for 15ma
  * `\uE511` (ottavaBassaBa) for 8vb
  * `\uE515` (quindicesimaBassa) for 15mb

### Phase 2: Adjust Layout Thresholds (`NotationCanvas.tsx`) — ✅ Completed
* **Objective:** Update the threshold math for `trebleShift` and `bassShift` and assign the new SMuFL glyphs.
* **Instructions:**
    1. Open `src/plugins/chord-notator/components/NotationCanvas.tsx`.
    2. Locate the `recalculateLayout` function (around line 430).
    3. Update the `trebleShift` logic to match the new step thresholds (18 for G6, 25 for G7):
       ```typescript
       let trebleShift = 0;
       let trebleLabelText: string | null = null;
       if (trebleNotesRaw.length > 0) {
         const maxTrebleStep = Math.max(...trebleNotesRaw.map(n => n.stepOffset));
         if (maxTrebleStep >= 25) { trebleShift = -14; trebleLabelText = SMuFL.quindicesimaAlta; }
         else if (maxTrebleStep >= 18) { trebleShift = -7; trebleLabelText = SMuFL.ottavaAlta; }
       }
       ```
    4. Update the `bassShift` logic to match the new step thresholds (-15 for B1, -22 for B0):
       ```typescript
       let bassShift = 0;
       let bassLabelText: string | null = null;
       if (bassNotesRaw.length > 0) {
         const minBassStep = Math.min(...bassNotesRaw.map(n => n.stepOffset));
         if (minBassStep <= -22) { bassShift = 14; bassLabelText = SMuFL.quindicesimaBassa; }
         else if (minBassStep <= -15) { bassShift = 7; bassLabelText = SMuFL.ottavaBassa; }
       }
       ```
