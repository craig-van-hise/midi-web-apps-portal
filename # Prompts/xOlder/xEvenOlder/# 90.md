## 1. Project Context & Objectives
The previous implementation failed to trigger the 100% active color flare when a keyswitch was played. The root cause was a brittle conditional check that failed to intercept the global MIDI listener's highlight color (`#aa3bff`). The objective is to make the `updateKeyVisuals88` override absolute: if a key is mapped as a keyswitch, its assigned semantic color must universally override any incoming active color state.

## 2. Technical Decisions & Dependencies
* **Absolute Interception:** Remove the `color === '#3b82f6'` constraint. If `color` is truthy (meaning the key is being played) and `ksColor` exists in the dataset, `ksColor` wins immediately.

       ```typescript
       export const updateKeyVisuals88 = (note: number, color: string) => {
           const el = document.getElementById(`pk88-${note}`);
           if (!el) return;

           const isBlack = [1, 3, 6, 8, 10].includes(note % 12);
           const ksColor = el.dataset.ksColor;

           if (color) {
               // ABSOLUTE OVERRIDE: If it's a keyswitch, its color always wins when played.
               const finalColor = ksColor ? ksColor : color;
               
               el.style.backgroundColor = finalColor;
               el.style.boxShadow = `inset 0 -5px 10px rgba(0,0,0,0.1), 0 0 12px ${finalColor}`;
               if (isBlack) {
                   el.style.zIndex = '11';
                   el.style.borderBottom = '1px solid #000';
                   el.style.borderLeft = '1px solid #333';
                   el.style.borderRight = '1px solid #333';
                   if (ksColor) el.style.borderTop = `2px solid ${ksColor}`;
               }
           } else {
               // KEY IS RELEASED/IDLE
               if (ksColor) {
                   el.style.backgroundColor = isBlack 
                       ? `color-mix(in srgb, ${ksColor} 60%, #444444)` 
                       : `color-mix(in srgb, ${ksColor} 45%, #ffffff)`;
                   el.style.borderTop = `2px solid ${ksColor}`;
               } else {
                   el.style.backgroundColor = isBlack ? '#444' : '#fff';
                   el.style.borderTop = 'none';
               }
               el.style.boxShadow = 'none';
               if (isBlack) {
                   el.style.zIndex = '10';
                   el.style.borderBottom = '6px solid #000';
                   el.style.borderLeft = '1px solid #333';
                   el.style.borderRight = '1px solid #333';
               }
           }
       };
       ```
