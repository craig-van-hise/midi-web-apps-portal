# Product Requirements Prompt (PRP) — 🔵 Active

## 1. Project Context & Objectives
The previous iteration of the keyswitch visuals resulted in an overly faint idle color wash, and the keys are failing to flare with their full 100% semantic color when actively played. The objective is to increase the `color-mix` intensity for idle keyswitches and to strictly rewrite the `updateKeyVisuals88` function to intercept the default active highlight color and replace it with the keyswitch's assigned color.

## 2. Technical Decisions & Dependencies
* **Color Intensity:** Increase white key idle wash to `45%` (was 25%) and black key idle wash to `60%` (was 35%).
* **Active Flare Interception:** Completely replace `updateKeyVisuals88` in `Keyboard.tsx` to strictly read `el.dataset.ksColor` and override the default `#3b82f6` highlight.
* **Badge Sizing:** Lock the badges back to `14px x 14px` to ensure they never clip adjacent keys.

* **Instructions:**
    1. **Update `updateKeyswitchBadge` Idle Wash:**
       * Change the `backgroundColor` assignment to use stronger percentages:
         `keyEl.style.backgroundColor = isBlack ? \`color-mix(in srgb, \${hex} 60%, #444444)\` : \`color-mix(in srgb, \${hex} 45%, #ffffff)\`;`
       * Ensure the badge `div` styling strictly uses `width: '14px'`, `height: '14px'` to prevent horizontal clipping.
    2. **Rewrite `updateKeyVisuals88`:** Replace the entire exported function in `Keyboard.tsx` with the following code to guarantee the active flare logic:
       ```typescript
       export const updateKeyVisuals88 = (note: number, color: string) => {
           const el = document.getElementById(`pk88-${note}`);
           if (!el) return;

           const isBlack = [1, 3, 6, 8, 10].includes(note % 12);
           const ksColor = el.dataset.ksColor; // Extract stored keyswitch color

           if (color) {
               // KEY IS PRESSED/ACTIVE
               // If it's a keyswitch and the incoming color is the default blue, override it!
               const finalColor = (ksColor && color === '#3b82f6') ? ksColor : color;
               el.style.backgroundColor = finalColor;
               el.style.boxShadow = `inset 0 -5px 10px rgba(0,0,0,0.1), 0 0 12px ${finalColor}`;
               if (isBlack) {
                   el.style.zIndex = '11';
                   el.style.borderBottom = '1px solid #000';
                   el.style.borderLeft = '1px solid #333';
                   el.style.borderRight = '1px solid #333';
                   if (ksColor) el.style.borderTop = `2px solid ${ksColor}`;
               }
           } else {
               // KEY IS RELEASED/IDLE
               if (ksColor) {
                   // Apply stronger idle wash for mapped keyswitches
                   el.style.backgroundColor = isBlack 
                       ? `color-mix(in srgb, ${ksColor} 60%, #444444)` 
                       : `color-mix(in srgb, ${ksColor} 45%, #ffffff)`;
                   el.style.borderTop = `2px solid ${ksColor}`;
               } else {
                   // Reset unmapped keys
                   el.style.backgroundColor = isBlack ? '#444' : '#fff';
                   el.style.borderTop = 'none';
               }
               el.style.boxShadow = 'none';
               if (isBlack) {
                   el.style.zIndex = '10';
                   el.style.borderBottom = '6px solid #000';
                   el.style.borderLeft = '1px solid #333';
                   el.style.borderRight = '1px solid #333';
               }
           }
       };
       ```
