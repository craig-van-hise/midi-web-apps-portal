## 1. Root Cause Analysis
The audio graph initializes without errors, but the Worklet produces no sound. This is because the `SharedArrayBuffer` reference is severed between `App.jsx` and `engine.js`. `App.jsx` instantiates `midiRingBuffer`, but fails to pass `midiRingBuffer.array.buffer` down to `<MasterRompler>`. Consequently, `MasterRompler.jsx` calls `audioEngine.init()` with no arguments, leaving `sabBuffer` undefined. The Worklet never receives the memory reference via `postMessage`, so it cannot pull MIDI data.
