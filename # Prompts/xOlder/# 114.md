# Product Requirements Prompt (PRP) — ✅ Completed

## 1. Project Context & Objectives
The `StepSequencer` component requires two visual layout corrections based on user feedback:
1. **Staff Line Bleed:** The horizontal staff lines in the Clef column extend to the left of the primary system barline, visually colliding with the Grand Staff Brace.
2. **Empty Bar Clutter:** The placeholder SMuFL whole rest glyphs (`\uE4E3`) in empty sequencer bars cause visual clutter and need to be removed.

## 2. Technical Decisions & Dependencies
* **Target File:** `src/plugins/chord-notator/components/StepSequencer.tsx`
* **Logic Specification:**
  * Adjust the absolute positioning of the staff line containers within the Clef column to start at `left-[12px]` (matching the `system-left-edge` barline) instead of `w-full` from `left-0`.
  * Strip the ternary logic `renderedNotes.length === 0 ? <Rest/> : <Notes/>` from the sequence map, replacing it with a simple direct `.map` over `renderedNotes` so empty arrays naturally render nothing.

## 3. Implementation Phases

### Phase 1: Constrain Clef Staff Lines (`StepSequencer.tsx`) — ✅ Completed
* **Objective:** Prevent staff lines from extending left of the system barline.
* **Instructions:**
    1. Open `src/plugins/chord-notator/components/StepSequencer.tsx`.
    2. Locate the `Clef & Brace Column` container.
    3. Find the `Staff Lines` section inside it. Replace the two `w-full` wrapper `div`s with `left-[12px]` and `right-0` so they span from the barline to the right edge:
       ```jsx
       {/* Staff Lines */}
       <div className="absolute right-0 left-[12px]" style={{ top: `calc(50% - ${MINI_STAFF * 6}px)` }}>
         {[0, 1, 2, 3, 4].map(i => <div key={i} className="w-full border-t border-black dark:border-gray-600 absolute opacity-60" style={{ top: `${i * MINI_STAFF}px` }} />)}
       </div>
       <div className="absolute right-0 left-[12px]" style={{ top: `calc(50% + ${MINI_STAFF * 2}px)` }}>
         {[0, 1, 2, 3, 4].map(i => <div key={i} className="w-full border-t border-black dark:border-gray-600 absolute opacity-60" style={{ top: `${i * MINI_STAFF}px` }} />)}
       </div>
       ```

### Phase 2: Remove Placeholder Rests (`StepSequencer.tsx`) — ✅ Completed
* **Objective:** Remove the SMuFL whole rests from empty sequence bars.
* **Instructions:**
    1. Scroll down to the `{/* 8 Bars Sequence */}` section.
    2. Locate the `{/* Notes or Rest */}` comment and the ternary block beneath it (`{renderedNotes.length === 0 ? (...) : (...)}`).
    3. Delete the ternary check and the entire `div` containing the whole rest (`{'\uE4E3'}`).
    4. Leave only the `renderedNotes.map(...)` block to render notes when they exist. It should look exactly like this:
       ```jsx
       {/* Notes */}
       {renderedNotes.map((n, i) => (
         <div key={i} className="absolute z-10" style={{ 
           left: n.xOffset !== undefined ? `calc(50% + ${n.xOffset}px)` : '50%', 
           top: `calc(50% - ${n.y}px)`, 
           transform: 'translate(-50%, -50%)' 
         }}>
           <span className="transition-colors" style={{ fontFamily: 'Bravura', fontSize: `${MINI_STAFF * 4.2}px`, color: itemColor }}>{SMuFL.noteheadWhole}</span>
           {(n.accidental || n.forceAccidentalDisplay) && (
             <span className="absolute transition-colors" style={{ 
               left: n.accidentalLeft || `calc(-1.5 * ${MINI_STAFF}px)`, 
               top: '50%', 
               transform: 'translateY(-50%)', 
               fontFamily: 'Bravura', 
               fontSize: `${MINI_STAFF * 3}px`,
               color: itemColor
             }}>
               {n.accidental || SMuFL.accidentalNatural}
             </span>
           )}

           {/* Ledger Lines */}
           {(() => {
             const lines = [];
             const renderLedgerLine = (lineStep: number) => {
               const yOffset = (n.finalStep - lineStep) * (MINI_STAFF / 2);
               return (
                 <div 
                   key={`ledger-${n.note}-${lineStep}`}
                   className="absolute left-1/2 -translate-x-1/2 h-[1px] z-[-1] transition-colors"
                   style={{
                     width: `${MINI_STAFF * 2.5}px`,
                     top: `calc(50% + ${yOffset}px)`,
                     backgroundColor: darkItemColor
                   }}
                 />
               );
             };

             if (n.isTreble) {
               if (n.finalStep >= 12) {
                 for (let ls = 12; ls <= n.finalStep; ls += 2) lines.push(renderLedgerLine(ls));
               } else if (n.finalStep <= 0) {
                 for (let ls = 0; ls >= n.finalStep; ls -= 2) lines.push(renderLedgerLine(ls));
               }
             } else {
               if (n.finalStep >= 0) {
                 for (let ls = 0; ls <= n.finalStep; ls += 2) lines.push(renderLedgerLine(ls));
               } else if (n.finalStep <= -12) {
                 for (let ls = -12; ls >= n.finalStep; ls -= 2) lines.push(renderLedgerLine(ls));
               }
             }
             return lines;
           })()}
         </div>
       ))}
       ```
