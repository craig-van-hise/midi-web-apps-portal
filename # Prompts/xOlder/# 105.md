# Product Requirements Prompt (PRP) — 🔘 Pending

## 1. Project Context & Objectives
Implement a new `StepSequencer` component containing 8 miniature bars of grand staff notation. It will act as a step-recorder for chords. When armed via a "Record" button, it listens to physical MIDI inputs; once a chord is released (all keys up), it captures a snapshot of the active notation, calculates the chord symbol, inserts it into the active bar, and automatically advances a red playhead to the next bar. It automatically disarms after the 8th bar or upon pressing the `Escape` key. This component will be mounted directly between the `NotationCanvas` and the `Keyboard`.

## 2. Technical Decisions & Dependencies
* **Target Files:**
  * `src/plugins/chord-notator/components/StepSequencer.tsx` (New File)
  * `src/plugins/chord-notator/index.jsx` (Layout Wrapper)
* **Logic Specification:**
  * **Event Sniffing:** The sequencer will maintain a `lastSeenChord` ref updated via the `refresh: true` payload broadcast by `NotationCanvas`.
  * **Commit Trigger:** It will track physical MIDI NoteOn/NoteOff events to maintain a `keysDown` counter. When `keysDown` drops to `0` during recording, it commits the `lastSeenChord` to the current step and advances.
  * **Scale Factor:** The mini-staves will use an internal `--mini-staff` CSS variable set to `5px` (scaled down from the main `12px`), allowing all 8 bars to fit inside a single standard-width card.
  * **Rest Glyph:** Empty bars will display the SMuFL whole rest glyph (`\uE4E3`).

### Phase 1: Build the Sequencer Logic (`StepSequencer.tsx`) — ✅ Completed
* **Objective:** Create the component structure, state management, and MIDI recording state machine.
* **Instructions:**
    1. Create a new file: `src/plugins/chord-notator/components/StepSequencer.tsx`.
    2. Import necessary hooks and utilities:
       ```tsx
       import React, { useState, useEffect, useRef } from 'react';
       import { useMidi } from '../midi/MIDIProvider';
       import { SMuFL } from '../utils/notationMath';
       import { getChordSymbol } from '../utils/chordSpeller';
       ```
    3. Define the internal state and the MIDI listener within the component:
       ```tsx
       export const StepSequencer: React.FC = () => {
         const { keySignature, lut } = useMidi();
         const [isRecording, setIsRecording] = useState(false);
         const [currentStep, setCurrentStep] = useState(0);
         const [sequence, setSequence] = useState<Array<{notes: any[], symbol: string}>>(
           Array(8).fill({ notes: [], symbol: '' })
         );

         const isRecordingRef = useRef(isRecording);
         const stepRef = useRef(currentStep);
         const activeKeys = useRef(0);
         const lastSeenChord = useRef<any[]>([]);

         useEffect(() => { isRecordingRef.current = isRecording; }, [isRecording]);
         useEffect(() => { stepRef.current = currentStep; }, [currentStep]);

         // Global Escape Key
         useEffect(() => {
           const handleKey = (e: KeyboardEvent) => {
             if (e.key === 'Escape') setIsRecording(false);
           };
           window.addEventListener('keydown', handleKey);
           return () => window.removeEventListener('keydown', handleKey);
         }, []);

         // MIDI Event Sniffer & Step Recorder
         useEffect(() => {
           const handleMidi = (e: Event) => {
             const customEvent = e as CustomEvent;
             const { data, refresh, notes } = customEvent.detail || {};

             // Constantly mirror the beautifully formatted notation from the main canvas
             if (refresh && notes) {
               lastSeenChord.current = notes;
             }

             // Track physical keys to determine when a chord is "released"
             if (data && data instanceof Uint8Array && data.length >= 3) {
               const [status, , vel] = data;
               const isNoteOn = (status & 0xF0) === 0x90 && vel > 0;
               const isNoteOff = (status & 0xF0) === 0x80 || ((status & 0xF0) === 0x90 && vel === 0);

               if (isNoteOn) activeKeys.current++;
               if (isNoteOff) activeKeys.current = Math.max(0, activeKeys.current - 1);

               // Commit Trigger: Keys released while recording
               if (isNoteOff && activeKeys.current === 0 && isRecordingRef.current) {
                 const currentNotes = lastSeenChord.current;
                 if (currentNotes.length > 0) {
                   const pitches = currentNotes.map(n => typeof n === 'object' ? n.note : n);
                   const symbol = getChordSymbol(pitches, keySignature, lut);
                   
                   setSequence(prev => {
                     const next = [...prev];
                     next[stepRef.current] = { notes: [...currentNotes], symbol };
                     return next;
                   });

                   setCurrentStep(s => {
                     if (s >= 7) {
                       setIsRecording(false);
                       return 0; // Wrap around to start, but disarmed
                     }
                     return s + 1;
                   });
                 }
               }
             }
           };

           window.addEventListener('MIDI_MESSAGE_RECEIVED', handleMidi);
           return () => window.removeEventListener('MIDI_MESSAGE_RECEIVED', handleMidi);
         }, [keySignature, lut]);
       ```

### Phase 2: Render the Mini-Canvas layout (`StepSequencer.tsx`) — ✅ Completed
* **Objective:** Implement the JSX to render the record button, 8 bars, ledger lines, and chord pills.
* **Instructions:**
    1. Still in `StepSequencer.tsx`, append the following return statement. Notice the use of `--mini-staff` set to `5px` to elegantly shrink the layout while maintaining the exact same mathematical proportions as the main canvas:
       ```tsx
         const MINI_STAFF = 5;

         return (
           <div className="w-full max-w-[962px] bg-white dark:bg-[#111] p-3 rounded-lg shadow-xl border border-gray-200 dark:border-gray-800 flex items-center gap-4 select-none">
             
             {/* Record Button */}
             <button 
               onClick={() => setIsRecording(!isRecording)}
               className={`w-12 h-12 rounded-full border-2 flex items-center justify-center transition-colors flex-shrink-0 ${isRecording ? 'border-red-500 bg-red-500/10' : 'border-gray-300 hover:border-gray-400 bg-transparent'}`}
             >
               <div className={`w-4 h-4 rounded-full ${isRecording ? 'bg-red-500 animate-pulse' : 'bg-gray-400'}`} />
             </button>

             {/* The 8 Bars Sequence */}
             <div className="flex-1 flex border-l-2 border-black dark:border-gray-600 h-[140px] relative overflow-hidden bg-white dark:bg-[#0a0a0a]">
               {sequence.map((bar, idx) => (
                 <div key={idx} className="flex-1 flex flex-col relative border-r border-black/30 dark:border-gray-600/50">
                   
                   {/* Playhead Indicator */}
                   {isRecording && currentStep === idx && (
                     <div className="absolute inset-0 bg-red-500/10 border-l-2 border-r-2 border-red-500/50 z-0 pointer-events-none" />
                   )}

                   {/* Mini Grand Staff System */}
                   <div className="flex-1 relative w-full h-full" style={{ '--mini-staff': `${MINI_STAFF}px` } as React.CSSProperties}>
                     
                     {/* Treble Lines */}
                     <div className="absolute w-full" style={{ top: `calc(50% - ${MINI_STAFF * 6}px)` }}>
                       {[0, 1, 2, 3, 4].map(i => <div key={i} className="w-full border-t border-black dark:border-gray-600 absolute opacity-60" style={{ top: `${i * MINI_STAFF}px` }} />)}
                     </div>
                     
                     {/* Bass Lines */}
                     <div className="absolute w-full" style={{ top: `calc(50% + ${MINI_STAFF * 2}px)` }}>
                       {[0, 1, 2, 3, 4].map(i => <div key={i} className="w-full border-t border-black dark:border-gray-600 absolute opacity-60" style={{ top: `${i * MINI_STAFF}px` }} />)}
                     </div>

                     {/* Notes or Rest */}
                     {bar.notes.length === 0 ? (
                       <div className="absolute top-1/2 left-1/2 -translate-x-1/2 -translate-y-1/2 text-black dark:text-gray-300 opacity-60" style={{ fontFamily: 'Bravura', fontSize: `${MINI_STAFF * 4}px` }}>
                         {'\uE4E3'} {/* Whole Rest */}
                       </div>
                     ) : (
                       bar.notes.map((n, i) => {
                         const yOffset = n.stepOffset * (MINI_STAFF / 2);
                         return (
                           <div key={i} className="absolute left-1/2" style={{ top: `calc(50% - ${yOffset}px)`, transform: 'translate(-50%, -50%)', zIndex: 10 }}>
                             <span className="text-black dark:text-white" style={{ fontFamily: 'Bravura', fontSize: `${MINI_STAFF * 4.2}px` }}>{SMuFL.noteheadWhole}</span>
                             {n.accidental && (
                               <span className="absolute text-black dark:text-white" style={{ left: `-${MINI_STAFF * 2}px`, top: '50%', transform: 'translateY(-50%)', fontFamily: 'Bravura', fontSize: `${MINI_STAFF * 3}px` }}>{n.accidental}</span>
                             )}
                           </div>
                         );
                       })
                     )}
                   </div>

                   {/* Chord Symbol Pill */}
                   <div className="h-8 flex items-center justify-center bg-gray-50 dark:bg-[#1a1a1a] border-t border-gray-200 dark:border-gray-800 z-10 relative">
                     <span className="text-xs font-bold text-blue-500" style={{ fontFamily: "'Jost', sans-serif" }}>
                       {bar.symbol || '-'}
                     </span>
                   </div>
                 </div>
               ))}
               
               {/* Final Barline */}
               <div className="absolute right-0 top-0 bottom-8 w-[2px] bg-black dark:bg-gray-600" />
             </div>
           </div>
         );
       };
       ```
