# Product Requirements Prompt (PRP) — Sequencer Real-Time Sync & Record Playhead 🔘

## 1. Project Context & Objectives
We need to enhance the `StepSequencer` recording UX with two critical improvements:
1. **Record Start Position:** When the user clicks the "Record" button, the recording playhead should begin at the currently selected bar (if one is selected) instead of always defaulting to bar 1.
2. **Real-Time NoteOn Sync:** When playing a chord via a MIDI controller in record mode, the chord and notation must appear on the timeline *immediately* upon key down, perfectly syncing with the main notation canvas. Currently, it only appears on key up.

## 2. Technical Decisions & Dependencies
* **Target File:** `src/plugins/chord-notator/components/StepSequencer.tsx`.
* **Playhead Logic:** Modify the Record button's `onClick` handler to initialize `currentStep` with `selectedStep` (falling back to 0).
* **Real-Time Sync:** Hijack the `refresh && notes` block inside the `handleMidi` listener. If `isRecording` is true and `notes` are present, immediately overwrite the `currentStep` sequence state. 
* **NoteOff Advancement:** Modify the raw `isNoteOff` logic so it no longer writes to the sequence (since NoteOn already did), but instead checks if the current step has notes and advances the playhead.

## 3. Implementation Phases

### Phase 1: Record Start Position (`StepSequencer.tsx`) — 🔘
* **Objective:** Ensure the record playhead (`currentStep`) jumps to the selected bar when record is toggled on.
* **Instructions:**
  1. Locate the Record `<button>` element (around line 214, the one with the `animate-pulse` red circle).
  2. Update its `onClick` handler to:
     ```javascript
     onClick={() => {
       const nextIsRecording = !isRecording;
       setIsRecording(nextIsRecording);
       if (nextIsRecording) {
         // Start from selected step, or 0 if none selected
         setCurrentStep(selectedStep !== null ? selectedStep : 0);
         setSelectedStep(null); // Clear selection visually to avoid UX confusion
       } else {
         setSelectedStep(null);
       }
     }}
     ```

### Phase 2: Real-Time NoteOn Sync (`StepSequencer.tsx`) — 🔘
* **Objective:** Instantly render chords on the timeline upon key down, and advance the playhead on key release.
* **Instructions:**
  1. Locate the `handleMidi` function. Find the `if (refresh && notes)` block.
  2. Rewrite the bi-directional sync block to support real-time recording previews:
     ```javascript
     if (refresh && notes) {
       lastSeenChord.current = notes;
       const currentNotes = notes;
       const pitches = currentNotes.map((n: any) => typeof n === 'object' ? n.note : n);
       const symbol = pitches.length > 0 ? getChordSymbol(pitches, keySignature, lut) : '';

       if (isRecordingRef.current && currentNotes.length > 0) {
         // Real-time preview: Draw immediately on NoteOn
         setSequence(prev => {
           const next = [...prev];
           next[stepRef.current] = { notes: [...currentNotes], symbol };
           return next;
         });
       } else if (selectedStepRef.current !== null && !isRecordingRef.current) {
         // Existing bi-directional sync for selected step
         setSequence(prev => {
           const next = [...prev];
           next[selectedStepRef.current!] = { notes: [...currentNotes], symbol };
           return next;
         });
       }
     }
     ```
  3. Further down in `handleMidi`, locate the raw `isNoteOff && activeKeys.current === 0 && isRecordingRef.current` block.
  4. Replace its contents. Since the chord is already drawn, we only need to advance the playhead if notes actually exist in the current step:
     ```javascript
     if (isNoteOff && activeKeys.current === 0 && isRecordingRef.current) {
       // Check if the real-time sync actually recorded anything before advancing
       const stepHasNotes = sequenceRef.current[stepRef.current]?.notes?.length > 0;
       if (stepHasNotes) {
         setCurrentStep(s => {
           if (s >= 7) {
             setIsRecording(false);
             return 0; 
           }
           return s + 1;
         });
       }
     }
     ```
