## 1. Project Context & Objectives

The keyswitches are failing to maintain their active 100% color flare when physically held down. A race condition is occurring where `NotationCanvas` broadcasts a `refresh` event immediately after a keyswitch is pressed, causing `Piano88` to erroneously send an empty color string (`''`) and turn the key off because it is not recognized as a standard chord note. The objective is to introduce an isolated hardware tracker (`physicalKeysTracker`) in `Keyboard.tsx` that guarantees a keyswitch remains fully illuminated for the entire duration of the physical key press, immune to downstream refresh sweeps.

## 2. Technical Decisions & Dependencies

* **Hardware State Tracking:** Implement a global `Set<number>` at the module level of `Keyboard.tsx` that listens directly to raw `MIDI_MESSAGE_RECEIVED` events to track physical `noteOn` and `noteOff` states independently of React's reconciliation cycle.
* **Absolute Interception:** Update `updateKeyVisuals88` to query the `physicalKeysTracker`. If a key is mapped as a keyswitch AND is physically held down, it will force the `ksColor` to render, overriding any empty string commands sent by external components.

// Tracks physical hardware state to prevent React from prematurely turning off keyswitches
const physicalKeysTracker = new Set<number>();
if (typeof window !== 'undefined') {
    window.addEventListener('MIDI_MESSAGE_RECEIVED', (e: any) => {
        const data = e.detail?.data;
        if (data && data.length >= 3) {
            const [status, note, velocity] = data;
            if ((status & 0xF0) === 0x90 && velocity > 0) physicalKeysTracker.add(note);
            if ((status & 0xF0) === 0x80 || ((status & 0xF0) === 0x90 && velocity === 0)) physicalKeysTracker.delete(note);
        }
    });
}

```

export const updateKeyVisuals88 = (note: number, color: string) => {
    const el = document.getElementById(`pk88-${note}`);
    if (!el) return;

    const isBlack = [1, 3, 6, 8, 10].includes(note % 12);
    const ksColor = el.dataset.ksColor;
    const isPhysicallyHeld = physicalKeysTracker.has(note);

    // ABSOLUTE OVERRIDE: If it's a keyswitch AND physically held, it stays on.
    let finalColor = color;
    if (ksColor && isPhysicallyHeld) {
        finalColor = ksColor;
    } else if (ksColor && color && color !== '') {
        // Allow UI clicks to trigger it if not physically held
        finalColor = ksColor;
    }

    if (finalColor) {
        // ACTIVE STATE
        el.style.backgroundColor = finalColor;
        el.style.boxShadow = `inset 0 -5px 10px rgba(0,0,0,0.1), 0 0 12px ${finalColor}`;
        if (isBlack) {
            el.style.zIndex = '11';
            el.style.borderBottom = '1px solid #000';
            el.style.borderLeft = '1px solid #333';
            el.style.borderRight = '1px solid #333';
            if (ksColor) el.style.borderTop = `2px solid ${ksColor}`;
        }
    } else {
        // IDLE STATE
        if (ksColor) {
            el.style.backgroundColor = isBlack 
                ? `color-mix(in srgb, ${ksColor} 60%, #444444)` 
                : `color-mix(in srgb, ${ksColor} 45%, #ffffff)`;
            el.style.borderTop = `2px solid ${ksColor}`;
        } else {
            el.style.backgroundColor = isBlack ? '#444' : '#fff';
            el.style.borderTop = 'none';
        }
        el.style.boxShadow = 'none';
        if (isBlack) {
            el.style.zIndex = '10';
            el.style.borderBottom = '6px solid #000';
            el.style.borderLeft = '1px solid #333';
            el.style.borderRight = '1px solid #333';
        }
    }
};

```
