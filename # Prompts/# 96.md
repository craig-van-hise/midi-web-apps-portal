# Product Requirements Prompt (PRP) — ✅ Completed

## 1. Project Context & Objectives
Implement an "Option + Drag" feature allowing users to physically drag transformation buttons from the `TransformationsToolbar` and drop them onto specific keys on the virtual `Keyboard` to map them as keyswitches. This supplements the existing MIDI Learn functionality. We will use the native HTML5 Drag and Drop API to automatically generate the ghost drag image for visual feedback, and apply direct DOM styling to highlight the target keys on hover.

## 2. Technical Decisions & Dependencies
* **Core API:** HTML5 Drag and Drop (`draggable`, `onDragStart`, `onDragOver`, `onDragEnter`, `onDragLeave`, `onDrop`).
* **Target Files:** * `src/plugins/chord-notator/components/toolbar/TransformationsToolbar.tsx` (Drag Sources)
  * `src/plugins/chord-notator/components/Keyboard.tsx` (Drop Targets)
* **Logic Specification:** * Buttons will have `draggable={true}`. 
  * `onDragStart` will halt the drag via `e.preventDefault()` *unless* `e.altKey` (Option key) is held. 
  * The Drag Payload will strictly be JSON: `{ type: 'KEYSWITCH_DRAG', buttonId: id }`.

## 3. Implementation Phases

### Phase 1: Configure Drag Sources (Toolbar) — ✅ Completed
* **Objective:** Allow buttons to be picked up only when the Option key is held, embedding their ID into the drag payload.
* **Instructions:**
    1. Open `TransformationsToolbar.tsx`.
    2. Inside the `<button>` element returned by `renderArrowBtn`, add the following props:
       ```jsx
       draggable={true}
       onDragStart={(e) => {
         if (!e.altKey) {
           e.preventDefault();
           return;
         }
         e.dataTransfer.effectAllowed = 'copy';
         e.dataTransfer.setData('application/json', JSON.stringify({ type: 'KEYSWITCH_DRAG', buttonId: id }));
       }}
       ```
    3. Apply the exact same `draggable` and `onDragStart` props to the `<button>` element returned by `renderActionBtn`.
* **Status Update:** "Transition Phase status to 🔵 Active. Apply modifications. Update status to ✅ Completed before initiating subsequent phases."

### Phase 2: Configure Drop Targets (Keyboard) — ✅ Completed
* **Objective:** Turn all physical keys into drop zones that highlight on hover and assign the keyswitch on drop.
* **Instructions:**
    1. Open `Keyboard.tsx`.
    2. Update the `useMidi()` destructured variables at the top of `Piano88` to include `updateButtonConfig`:
       `const { dispatchVirtualMidi, lut, keySignature, selectedNotes, buttonConfigs, updateButtonConfig } = useMidi() as any;`
    3. Locate the White Key `<div>` (inside the `if (!isBlack)` block). Add the following DnD event handlers:
       ```jsx
       onDragOver={(e) => { e.preventDefault(); e.dataTransfer.dropEffect = 'copy'; }}
       onDragEnter={(e) => { e.currentTarget.style.outline = '3px solid #aa3bff'; e.currentTarget.style.outlineOffset = '-3px'; }}
       onDragLeave={(e) => { e.currentTarget.style.outline = ''; }}
       onDrop={(e) => {
           e.preventDefault();
           e.currentTarget.style.outline = '';
           try {
               const data = JSON.parse(e.dataTransfer.getData('application/json'));
               if (data.type === 'KEYSWITCH_DRAG' && data.buttonId) {
                   updateButtonConfig(data.buttonId, { midiNote: note });
               }
           } catch (err) {}
       }}
       ```
    4. Locate the Black Key `<div>` (inside the `hasRightBlack` conditional). Add the following DnD event handlers, ensuring `stopPropagation()` is used so it doesn't trigger the white key underneath:
       ```jsx
       onDragOver={(e) => { e.preventDefault(); e.stopPropagation(); e.dataTransfer.dropEffect = 'copy'; }}
       onDragEnter={(e) => { e.stopPropagation(); e.currentTarget.style.outline = '2px solid #aa3bff'; e.currentTarget.style.outlineOffset = '-2px'; }}
       onDragLeave={(e) => { e.stopPropagation(); e.currentTarget.style.outline = ''; }}
       onDrop={(e) => {
           e.preventDefault();
           e.stopPropagation();
           e.currentTarget.style.outline = '';
           try {
               const data = JSON.parse(e.dataTransfer.getData('application/json'));
               if (data.type === 'KEYSWITCH_DRAG' && data.buttonId) {
                   updateButtonConfig(data.buttonId, { midiNote: note + 1 });
               }
           } catch (err) {}
       }}
       ```
