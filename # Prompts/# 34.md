## 1. Project Context & Objectives
The VV | WebApps Portal is undergoing UI density improvements and logic refinements for the MIDI Transposer. The objective is to remove the deprecated `note-range-filter` plugin, reorder the sidebar, and significantly upgrade the MIDI Transposer's functionality. Specifically, we must implement a dynamic polyphony routing rule, bridge the UI keyboards so that mouse clicks generate actual MIDI events through the audio signal chain, update the default Transpose octave to +1, shift the Transpose UI color from red to orange, and scale down the vertical footprint of all virtual keyboards by 33%.

## 2. Technical Decisions & Dependencies
* **App Decommissioning:** Complete removal of `src/plugins/note-range-filter/` and its registry entry.
* **Synthetic MIDI Event Bus Routing:** To make the UI keyboards playable, we will not bypass `useWebMidi.js`. Instead, `index.jsx` will pass a new prop to the keyboards: `simulateMidi={(data) => midiBus?.dispatchEvent(new CustomEvent('midi', { detail: data }))}`. The keyboard's `onMouseDown`/`onMouseUp` handlers will fire this, tricking the plugin into processing the click exactly like hardware input.
* **Polyphony Logic Constraint:** The rule for forcing the Play Zone into Mono must be evaluated dynamically in the MIDI processing hook (`useWebMidi.js`).
  * **Formula:** `const forceMono = store.polyphonyMode === 'poly' && store.transposeTargets.length > 1;`
  * If `forceMono` is true, overlapping notes in the Play Zone must send Note-Offs for previous notes to enforce monophony. If false, the Play Zone operates fully polyphonically.
* **Color Map Adjustments:** Replace `#f43f5e` (Red) with `#f97316` (Orange) in `useMidiStore.js`. Update any corresponding Tailwind classes in the transposer components.
* **Keyboard Scaling:** Decrease the height of `keyboard-wrapper`, white keys, and black keys by exactly 33% in all keyboard components.
  * White keys: `88px` -> `58px`.
  * Black keys: `56px` -> `37px`.
  * Wrapping containers: Scale proportionately.
